# Lantern City Documentation Workspace

Dedicated working directory for all Lantern City design materials.

Root structure:
- briefs/      → master design docs and summaries
- districts/   → district-by-district notes
- factions/    → faction profiles and agendas
- mechanics/   → rules, systems, loops, and state tracking
- npcs/        → important NPC sheets and relationships
- sessions/    → session notes, scene drafts, and running story logs
- notes/       → loose ideas and scratch notes
